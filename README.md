# Restoran
# Žar

Žar je statička web stranica restorana izrađena za Projekt 1 iz kolegija Uvod u Web tehnologije. Web prikazuje osnovne informacije o restoranu, jelovnik, dostavu, lokaciju, rezervacije i kontakt.

## Stranice

- `index.html` - naslovnica
- `jelovnik.html` - ponuda pića, predjela, glavnih jela i deserata
- `dostava.html` - informacije o dostavi
- `lokacija.html` - adresa, karta i radno vrijeme
- `rezervacije.html` - forma za rezervaciju stola
- `onama.html` - informacije o restoranu
- `kontakt.html` - kontakt podaci restorana

## Kako pokrenuti

Otvorite `index.html` u pregledniku  
ili koristite **Live Server** u VS Code-u.

## Tehnologije

- HTML5
- CSS3
- Flexbox
- Grid
- CSS varijable
- Responsive dizajn
- Hamburger izbornik bez JavaScripta

## Značajke

- globalna navigacija na svim stranicama
- aktivna stavka u navigaciji
- responsive dizajn za mobitel i desktop
- pristupačnost:
  - skip link
  - vidljiv focus
  - alt atributi na slikama
- forma s HTML validacijom
- SEO meta tagovi i Open Graph tagovi
